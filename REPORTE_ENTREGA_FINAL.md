## Funcionalidades entregadas

- Gestión de inventario con datos semilla.
- Histórico de ventas por SKU.
- Cálculo de demanda promedio.
- Inventario de seguridad.
- Punto de reorden.
- Días de cobertura.
- Alertas de inventario.
- Clasificación ABC.
- Pronóstico por media móvil.
- Pronóstico por suavizamiento exponencial.
- Machine Learning supervisado con Random Forest.
- Machine Learning no supervisado con KMeans.
- Dashboard ejecutivo.
- Exportación CSV.
- Pruebas unitarias.
- Smoke test.
- Preparación para Render.
